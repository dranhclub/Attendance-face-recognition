# Ứng dụng điểm danh bằng nhận diện khuôn mặt
Bài tập lớn môn Thị giác máy tính

## Công nghệ sử dụng
- Thư viện [face_recognition](https://github.com/ageitgey/face_recognition)
- Xây dựng website bằng Flask


## Requirements
- `pip install face_recognition`
- `pip install Flask`

## How to run
- `python app.py`
